## Italiano

### Installazione

1. Aprite [claude.com/plugins](https://claude.com/plugins) e cercate **BetterCallClaude**
2. Cliccate **Installa** — il plugin si aggiunge automaticamente al vostro Cowork Desktop
3. Condividete una cartella con il vostro fascicolo o i documenti da analizzare

### Primo utilizzo

Digitate nella chat:

```
/bettercallclaude:start
```

BetterCallClaude vi guiderà passo passo:
- Verifica della connessione ai servizi legali
- Creazione del vostro **playbook locale** (preferenze dello studio/reparto legale)
- Esempi di utilizzo personalizzati per il vostro profilo

### Esempi rapidi

| Cosa volete fare | Cosa scrivere |
|------------------|---------------|
| Analizzare un NDA | "Analizza questo NDA e dimmi se è accettabile" |
| Controllare più NDA | "Controlla tutti gli NDA nella cartella condivisa" |
| Ricerca giurisprudenziale | "Cerca sentenze sulla disdetta anticipata del contratto di locazione" |
| Analisi completa di una pratica | "Analizza questa pratica dall'intake alla bozza di Klageschrift" |
| Tradurre un parere | "Traduci questo parere in francese" |

I risultati vengono salvati come file nella vostra cartella condivisa (sottocartella `bcc-output/`).

### Mini-glossario

| Termine | Significato |
|---------|-------------|
| **Comando** | Un'istruzione che inizia con `/bettercallclaude:` — attiva una funzione specifica |
| **Skill** | Conoscenza specializzata che si attiva automaticamente quando serve |
| **Server MCP** | Servizio che accede a banche dati giuridiche ufficiali (BGE, Fedlex, ecc.) |
| **Playbook** | File di configurazione con le preferenze del vostro studio |

## Deutsch

### Installation

1. Öffnen Sie [claude.com/plugins](https://claude.com/plugins) und suchen Sie **BetterCallClaude**
2. Klicken Sie auf **Installieren** — das Plugin wird automatisch zu Ihrem Cowork Desktop hinzugefügt

Geben Sie im Chat ein:


BetterCallClaude führt Sie Schritt für Schritt:
- Prüfung der Verbindung zu den Rechtsdatenbanken
- Erstellung Ihres **lokalen Playbooks** (Kanzlei-/Rechtsabteilungspräferenzen)

| Was Sie tun möchten | Was Sie schreiben |

| Begriff | Bedeutung |

## English

### Installation

1. Open [claude.com/plugins](https://claude.com/plugins) and search for **BetterCallClaude**
2. Click **Install** — the plugin is automatically added to your Cowork Desktop
3. Share a folder containing your case files or documents to analyze

### First use

Type in the chat:

```
/bettercallclaude:start
```

BetterCallClaude will guide you step by step:
- Check connectivity to Swiss legal databases
- Create your **local playbook** (firm/legal department preferences)
- Personalized usage examples for your profile

### Quick examples

| What you want to do | What you type |
|---------------------|---------------|
| Analyze an NDA | "Analyze this NDA and tell me if it's acceptable" |
| Check multiple NDAs | "Check all NDAs in the shared folder" |
| Case law research | "Find case law on early lease termination" |
| Full case analysis | "Analyze this case from intake to draft Klageschrift" |
| Translate a legal opinion | "Translate this opinion to French" |

Results are saved as files in your shared folder (`bcc-output/` subfolder).

### Mini-glossary

| Term | Meaning |
|------|---------|
| **Command** | An instruction starting with `/bettercallclaude:` — activates a specific function |
| **Skill** | Specialized knowledge that activates automatically when relevant |
| **MCP server** | Service accessing official Swiss legal databases (BGE, Fedlex, etc.) |
| **Playbook** | Configuration file with your firm's preferences |
